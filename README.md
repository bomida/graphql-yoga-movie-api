# graphql-yoga-movie-api

## 소개
GraphQL-Yoga를 이용해 영화 앱을 위한 Movie API를 만들어보았습니다.

<br/>

## 사용기능
### NodeJS
  - GraphQL Yoga, Nodemon, Babel
### GraphQL Yoga
### babel
### nodemon
  - nodemon을 설치하여 파일을 수정할 때마다 서버를 자동으로 재시작 해주어 코드 작성 시 오류를 확인 할 수 있었습니다.
